# MCP (Model Context Protocol) 활용 가이드

본 프로젝트에서는 AI 모델의 외부 데이터 연동 및 실행 능력을 극대화하기 위해 다음 5가지 MCP 도구를 활용합니다.

## 1. Brave Search MCP
* **목적**: 최신 시장 트렌드, 편의점 신상품 뉴스, 경쟁사 동향 파악
* **주요 기능**: 웹 검색 및 관련 기사 URL 추출
* **필요 설정**: `BRAVE_API_KEY` 발급 필요

## 2. Fetch MCP
* **목적**: 뉴스 기사, 블로그 포스트 등의 텍스트 데이터 수집
* **주요 기능**: 정적 웹페이지를 마크다운 형식으로 변환하여 모델에 제공

## 3. Puppeteer MCP
* **목적**: 동적 웹사이트(인스타그램, 배달 앱 등)에서 트렌드 데이터 수집
* **주요 기능**: 브라우저 자동화를 통한 스크린샷 캡처 및 DOM 데이터 추출

## 4. Python Executor MCP
* **목적**: 로컬 환경에서의 안전한 데이터 연산 및 시각화 테스트
* **주요 기능**: GNN 텐서 차원 검증, EDA 스크립트 실행, 가상 샌드박스 연산

## 5. GitHub MCP
* **목적**: 소스 코드 관리 및 품질 보증 자동화
* **주요 기능**: PR 리뷰, 이슈 관리, 하네스(GEMINI.md) 위반 사항 체크 및 자동 수정 제안

---
**주의**: GitHub Personal Access Token이 `mcp_config.json`에 포함되어 있습니다. 보안을 위해 해당 파일이 공용 저장소에 push되지 않도록 주의하십시오.
